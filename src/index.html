Welcome

To injest data:
  • from browser http://localhost:5000/api/irs/ingest
  • from shell: curl http://localhost:5000/api/irs/ingest
To view sample data:
  • from browser http://localhost:5000/api/irs/sample
  • from shell: curl http://localhost:5000/api/irs/sample
To view columns:
  • from browser http://localhost:5000/api/irs/column
  • from shell: curl http://localhost:5000/api/irs/column
To query data:(only support limited query options for demo purpose)
  • from browser http://localhost:5000/api/irs/data?max=10 (retrieve 10 records)
  • from shell: curl http://localhost:5000/api/irs/data
Parameter Description Example
q Data query q={"tax_period": {"eq": "201109"}}
max Maximum number of records retrieved max=10
Operator Description Example
eq = {"tax_period": {"eq":"201106"}}
gt > {"tax_period": {"gt":"201106"}}
lt < {"tax_period": {"lt":"201106"}}
in Match any value in array {"tax_period": {"in":["201106","201109"]}}

Query Example

Query Description
http://localhost:5000/api/irs/data retrieve all data
http://localhost:5000/api/irs/data?q={"tax_period":{"eq":"201106"}}&max=10 retreive 10 records which tax_period equal to "201106"
http://localhost:5000/api/irs/data?q={"tax_period":{"in":["201106","201109"]}} retrieve records which tax_period equal to "201106" or "201109"
